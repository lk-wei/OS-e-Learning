Shortest Job First - Tutorial

You will play as a bus conductor and it is your job to assign the order in which the passengers will board the bus using the logic of the selected algorithm Shortest Job First.


The Shortest Job First algorithm assigns the process with the shortest burst time to the CPU first.


In the game, passengers will arrive in Groups. Each group represents a process. And the Amount of passengers in each groups represents the burst time of each process.



Board the group with the least passengers first and so on.
